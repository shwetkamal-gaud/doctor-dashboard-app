## Installation

### 1. Clone the Repository
```bash
git clone https://github.com/shwetkamal-gaud/doctor-dashboard-app.git
cd doctor-dashboard-app
```

### 2. Install Dependencies 
#### Backend:
```bash
cd backend
npm install
```
#### Frontend
```bash
cd frontend
npm install
```

### 3. Run the app
```bash
cd frontend
npm start
```
